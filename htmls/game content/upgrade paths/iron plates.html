Iron Plates
General

Iron Plates is an upgrade path of the Tower. It can be chosen upon upgrading a tower to level 3 and gives the tower a massive amount of health.

Description

This tower becomes an armored fortress, that will not go down easily.

Details

The damage dealt is calculated with this formular: Base Damage x Type Value 1 x Type Value 2 x ... x Type Value n, where n is the amount of types that the enemy has (all types that are not provided in the table above have a Value of 1).
