Lightning Desert

Every couple of months, the developers host an official Thronefall tournament! Players can participate on the official Thronefall discord server.

Lightning Desert was the first official tournament that has been hosted. The developers wanted to keep it simple, so everyone could participate. The goal of the challenge was to achieve the highest score possible while playing on the map durststein using the lightning staff weapon and commander mode perk.

The start of the event was the 6th October 2023 and it ended on the 13th of the same month.
