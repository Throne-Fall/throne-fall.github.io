Shadow Codex

General

The Shadow Codex is the fifth available weapon in the game

It curses enemies and deals damage to every cursed enemy at once.
Its active ability removes all curses to deal a big amount of.

Details

Active Ability

Statistic Value

The damage dealt is calculated with this formula: Base Damage x Type Value 1 x Type Value 2 x ... x Type Value n, where n is the amount of types that the target has (all types that are not provided in the table above have a Value of 1).
