Support Tower

Removed feature

This feature is only available in versions before 1.5.

General

Support Tower is an upgrade path of the Tower. It can be chosen upon upgrading a tower to level 3 and enables the tower to heal allied units and the King. This comes at the cost of no longer dealing damage to enemies.

Description

This tower no longer shoots at enemies. Instead it shoots healing orbs at you and your units.
